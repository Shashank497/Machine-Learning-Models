# Gold Price Prediction

This project predicts gold prices using historical data and machine learning techniques. The goal is to analyze price trends and build a model that can estimate future gold prices.

---

## Project Files

- `GPP.ipynb` – Jupyter Notebook containing data analysis, visualization, and model training
- `gld_price_data.csv` – Dataset with historical gold price information
- `README.md` – Project documentation

---

## Technologies Used

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

## What This Project Does

- Loads and preprocesses gold price data  
- Performs exploratory data analysis  
- Trains a machine learning model  
- Evaluates model performance  
- Predicts gold prices based on historical data
